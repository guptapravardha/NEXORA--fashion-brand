# **App Name**: NEXORA

## Core Features:

- Cinematic Storytelling Engine: Utilizes GSAP and ScrollTrigger for high-end, editorial scroll experiences including parallax imagery and staggered text reveals for a premium brand feel.
- NEXORA Stylist AI: An AI-powered tool that analyzes browsing patterns to suggest complete curated looks; the LLM uses reasoning as a tool to determine which pieces complement each other for specific occasions.
- INR-Localized Catalog: A robust catalog of 30+ luxury items categorized by gender (For Her, For Him) with pricing in Indian Rupees and seamless category filtering.
- Luxury Checkout Orchestrator: A sophisticated multi-step checkout flow including shipping, delivery preferences, and order summary with localized currency management.
- Persistent Luxury Cart: A high-performance client-side cart system using LocalStorage for persistence and real-time state management.
- Razorpay Production Bridge: Modular architecture for secure Indian Rupee (INR) transactions, pre-configured for production environment variables.

## Style Guidelines:

- Primary palette: Deepest Black (#000000) and Optic White (#FFFFFF) with Soft Pearl Grey backgrounds for a cinematic editorial canvas.
- Accent: Burnished Gold (#B5935A) used sparingly for high-end call-to-action buttons and premium highlights.
- Headline: High-contrast Playfair Display serif for an elegant, high-end fashion authority.
- Body: Minimalist PT Sans sans-serif for readable product descriptions and technical details.
- Ultra-thin 2px stroke icons to maintain a minimal, Apple-level polish across all navigation elements.
- Modular clean grids with intentional editorial whitespace and oversized high-resolution product imagery.
- Extensive GSAP ScrollTrigger implementations for parallax effects, sticky navigation transitions, and smooth section reveals.
